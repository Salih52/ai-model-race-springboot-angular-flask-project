Enter Pre-Process Code

Eğer Modelini eğitmeden önce eğitim versinde ön işleme yapıtysanız ön işleme methodunuzu buraya yapıştırınız.Eğer yapmadıysanız burayı boş bırakıp modelinizi direk yükleyin. Methodu yazarken aşağıdaki kurallara lütfen uyunuz eğer method bunlara uygun değilse veya methodunuzu buraya koymassanız modeliniz çalışmayacaktır

1 - Ön İşleme yapılacak veri setinin adı "df" olucaktır

2 - Methodunuzun adı "preprocessing" olmak zorundadır

3 - Eğer bir şey import ediyor iseniz bunu da ekleyin

4 - İzin verilen kütüphaneler: pandas, numpy, sklearn

5 - Methodunuzun sonucunu geri döndürmeyi unutmayın

örnek Code: 

          
            from sklearn.preprocessing import StandardScaler
            def preprocessing(df):
                x = df.iloc[:, :10]  # İlk 10 özelliği seçiyoruz
                x['memory_battery'] = x['memory'] * x['battery']
                scaler = StandardScaler()
                x = scaler.fit_transform(x)
                return x
